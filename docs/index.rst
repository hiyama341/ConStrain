Welcome to ConStrain's documentation!
======================================

The mission for is  is to revolutionize the way in wich we make biology. How do we do that? 
We pursue reproducible high-throughput strain construction by adopting literate programming with ConStrain. 
We hope you want to be part of the revolution by adopting this way of doing biology. 
Below you will find documentation for how to install and use ConStrain.

If you are interested in examples, head over to our colab_notebooks. 
Enjoy and reach out if something is unclear. Lets start the revolution together.
